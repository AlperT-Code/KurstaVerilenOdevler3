HTML Nedir?

HTML (HyperText Markup Language), web sayfalarının yapısını oluşturmak için kullanılan bir işaretleme dilidir. HTML, metin, görseller, bağlantılar, formlar ve diğer içeriklerin düzenlenmesini sağlar. HTML, etiketler (tags) kullanılarak oluşturulur ve tarayıcılar tarafından işlenir.

Örnek HTML etiketi: <div>...</div>

CSS Nedir?

CSS (Cascading Style Sheets), web sayfalarının görünümünü ve düzenini belirlemek için kullanılan bir stil dilidir. CSS, HTML elemanlarına stil uygulamak için kullanılır ve renk, yazı tipi, kenar boşlukları, hizalama gibi görsel özellikleri kontrol eder.

Örnek CSS kodu:

body {
    background-color: #f4f4f4;
    font-family: Arial, sans-serif;
}

JavaScript Nedir?

JavaScript, web sayfalarına dinamik içerik eklemek için kullanılan bir programlama dilidir. Kullanıcı etkileşimi, animasyonlar, form doğrulamaları ve daha fazlası JavaScript ile yapılabilir. JavaScript, HTML ve CSS ile birlikte çalışarak etkileşimli ve dinamik web sayfaları oluşturulmasını sağlar.

Örnek JavaScript kodu:

function greet() {
    alert('Merhaba Dünya!');
}
